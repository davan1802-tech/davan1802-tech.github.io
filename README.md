# danganhvan.github.io

A personal resume website for Dang Anh Van, Senior Java Backend Developer. This site showcases professional experience, technical skills, education, and certifications in a clean, responsive, and printable format.

## Features

- Modern, two-column resume layout inspired by Travis Willis style
- Responsive design for desktop and mobile
- Downloadable PDF CV button
- Print-friendly formatting
- Clean, semantic HTML and CSS

## Technologies Used

- HTML5
- CSS3 (custom, no frameworks)
- Google Fonts (Playfair Display, Lora)

## Getting Started

To run locally:

1. Clone the repository:
   ```sh
   git clone https://github.com/vanpersieyolo/danganhvan.github.io.git
   cd danganhvan.github.io
   ```
2. Open `index.html` in your web browser.

To deploy, simply push to the `main` branch of your GitHub repository. GitHub Pages will serve the site automatically.
